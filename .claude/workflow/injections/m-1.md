# M-1 Explore 注入清单（探索优先，比例仅作参考）

目标：判断素材能否形成值得验证的产品方向，而不是交付生产功能。

必须读取：

- `.rpi-outfile/product/sources.json`
- `.rpi-outfile/product/claims.json`
- 当前分析、原型或研究记录

允许：

- 产品原型、能力原型、工程 Spike
- 模拟数据和可抛弃实现
- 多个候选方向并行比较

禁止：

- 将 AI 推断直接写成产品事实
- 将探索代码未经正式 Spec + TDD 直接升级为生产代码
- 以“仍是原型”为由绕过隐私、安全和不可逆操作底线

退出证据：明确用户、核心问题、候选主链路、关键能力可行性和主要风险。
